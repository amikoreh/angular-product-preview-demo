Simple Angular Product Preview App Demo
================

#### Application Requires/Configuration:
* Angular 2
* Node 4.6.0
* npm 4.6.1
* Typescript 2.3.3
* Typings 1.5
* Lite Server 2.3.0

##### Build + Install:
* `npm install`
    
##### Run
* `npm start`
* Open at http://localhost:3000/
