Simple Angular Product Preview App Demo
================

#### Application Requires/Configuration:
* Angular 2
* Node 4.8.4 or Node 6.11.2
* npm 5.3.0
* Typescript 2.3.3
* Typings 1.5
* Lite Server 2.3.0

##### Build + Install:
* `npm install`
* `npm update`
    
##### Run
* `npm start`
* Open at http://localhost:3000/
